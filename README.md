# Kalshi Longshot BiasReproducible research + live EV scanner for Kalshi longshot bias. Defaults to research-only (`DRY_RUN=True`) and does not place orders.## Setup- Install deps: `pip install -r requirements.txt`- Optional env vars:  - `KALSHI_ACCESS_KEY` (access key id)  - `KALSHI_PRIVATE_KEY_PATH` (path to RSA private key PEM)  - `KALSHI_PRIVATE_KEY_PEM` (PEM contents as env var)  - `KALSHI_SSL_VERIFY=false` if your network intercepts TLS  - `KALSHI_CANDLE_INTERVAL_SEC` (candlestick interval, default `60`)  - `KALSHI_DEFAULT_FEE_MULTIPLIER` (default `0.07`)  - `DRY_RUN=false` (only relevant if order placement is added later)## Run```bashpython -m kalshi_longshot_bias.cli run-all --start 2024-01-01 --end 2024-01-07```Full history (resumable):```bashpython -m kalshi_longshot_bias.cli run-all --all-history --resume```Outputs:- `outputs/bias_stats.csv`- `outputs/positive_ev_bets.csv`- `outputs/report.md`- `outputs/plots/*.png`Paper trading (resumable, research-only):```bashpython -m kalshi_longshot_bias.cli paper-trade --initial-cash 1000 --stake 10 --poll-seconds 60```Windows launcher with log file:```powershell.\run_paper_trading.ps1 -InitialCash 1000 -Stake 10 -PollSeconds 60```One-loop smoke test:```bashpython -m kalshi_longshot_bias.cli paper-trade --once --max-markets 200```Paper-trading artifacts (continuously updated under `outputs/paper_trading/`):- `state.json` (resume checkpoint)- `events.jsonl` (append-only event log)- `orders.csv`- `fills.csv`- `closed_trades.csv`- `equity_curve.csv`- `open_orders.csv`- `open_positions.csv`- `scan_summary.csv`- `candidates_latest.csv`Raw cache:- `data/raw/*.jsonl`
